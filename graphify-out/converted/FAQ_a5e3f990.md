## Sheet: Sheet1
| Sr No | Question | Answer |
| --- | --- | --- |
|  | 🔹 Parent Communication & Scope |  |
| 1 | What should a parent do if they email the support address with a general school-related question? | The support email support@uniqueschools.ie handles app/technical issues only. If a general school-related query is received, the support team will direct the parent to contact the school directly. Parents should not use the support email for general school matters. |
| 2 | How can parents contact the school directly instead of using the support email? | Parents should contact the school through the school's own contact channels (phone, school email, or in person). The App's Note from Parent feature can also be used to communicate directly with the school. |
|  | 🔹 Attendance Queries |  |
| 3 | Can attendance-related notes (absence, late, permission to leave) be submitted via the support email? | No. Attendance notes must be submitted through the App's Digital Attendance Forms section. The support email is not the correct channel for attendance notifications. |
| 4 | What is the correct way to submit attendance-related notes? | Through the App's Digital Attendance Forms (Absentee Notes, Late Notes, Permission to Leave Early, Return after Absence). Forms must be submitted before 08:15am to allow school staff time to process them. An optional 4-digit PIN security can be enabled for added security. |
|  | 🔹 VSware / Tyro Access Issues |  |
| 5 | What should a user do if their VSware username is not auto-filled in the app? | Email support@uniqueschools.ie. The support team will investigate why the username is not auto-populating in the App. |
| 6 | How can a user reset their VSware password from within the app? | Tap VSware on the App homescreen → tap "Create or Reset Password" → enter Username (copy from the first screen) and the last 4 digits of your mobile number → tap Send Code → enter the SMS code received → enter a new password → tap Change Password. |
| 7 | What should a user do if they do not have a Tyro username? | Contact support@uniqueschools.ie. The support team will liaise with the school to ensure the correct Tyro credentials are set up and linked to the user's profile. |
| 8 | How can a user resolve a locked VSware account? | Tap the VSware Help icon (top right of the VSware screen in the App) → tap "No" if unable to log in → access the Help Guide or tap "Email Support" to describe the issue and submit it to the support team at support@uniqueschools.ie. |
| 9 | What should a user do if they do not receive a VSware OTP verification code? | Wait for the countdown timer to expire and then request the code again. Ensure mobile signal is available. If the issue persists, contact support@uniqueschools.ie with details of the problem. |
| 10 | How can a user fix a blank screen issue in VSware? | Close and reopen the App. If the issue persists, try tapping "Manage Autofill" to re-enter and save VSware credentials. If still unresolved, use the VSware Help icon (top right) to email support at support@uniqueschools.ie. |
| 11 | Can a parent access multiple VSware accounts across different schools from a single app? | Yes. The App supports saving multiple sets of VSware credentials through the "Manage Autofill" settings, allowing different usernames and passwords to be saved for each student/school. |
|  | 🔹 Registration Issues |  |
| 12 | What does the error "mobile number not found in the system" mean during registration? | It means the mobile number entered has not been registered with the school in their administration system. The school must first add the parent/guardian's mobile number to their system before registration can proceed. |
| 13 | How can a user resolve the "mobile number not found" issue? | Contact the school directly and ask them to add or update your mobile number in their administration system. Once the school updates the record, you can retry registration. |
| 14 | What does the error "mobile number already taken" indicate? | The mobile number is already linked to an existing account in the App. Another user (or a previous registration attempt) has already registered with that mobile number. |
| 15 | What should a user do if their mobile number is already registered? | Contact support@uniqueschools.ie. The support team will investigate and help resolve the duplicate registration issue. |
| 16 | What does the error "email address already registered" mean? | The email address entered is already linked to an existing account in the App. The user may have previously registered with that email. |
| 17 | What should a user do if their email is already registered? | Use the "Forgot Password" option on the login screen to regain access to the existing account. Alternatively, contact support@uniqueschools.ie for assistance. |
| 18 | Can Unique Schools update or modify user data in the school system? | No. Unique Schools does not have access to modify data in the school's administration system (e.g., VSware). Only the school's own administration staff can update student/parent records. Any data corrections must be requested directly from the school. |
|  | 🔹 Login & Account Management |  |
| 19 | How can a user change their registered email address in the app? | Open the App homescreen → tap your name → tap "Change Email ID" → enter the new email address and your current password → verify the new email by clicking the link sent to the new inbox (check Spam/Junk if not found) → tap "I have verified my email ID" to complete the change. |
| 20 | How can a user change their registered phone number? | Contact support@uniqueschools.ie. The support team will coordinate with the school to update the mobile number in the school's system, after which the App record can be updated. |
| 21 | What steps should a user follow if they forget their password? | 1. Open the App and tap "Forgot Password" → 2. Enter the registered email address → 3. Tap "Reset Password" → 4. Open the verification email sent to that address → 5. Click the verification link → 6. Enter and save the new password → 7. Return to the login screen → 8. Accept Terms & Conditions → 9. Log in with the new credentials. |
| 22 | How can a user reset or disable their PIN in the app? | Go to your Profile settings in the App and navigate to the PIN/Security section to reset or disable the 4-digit PIN. If you have trouble, tap "Having a Problem?" or contact support@uniqueschools.ie. |
|  | 🔹 Payments Issues |  |
| 23 | What does a failed EMI transaction notification indicate? | It indicates that a scheduled instalment (EMI) payment attempt was unsuccessful — the system was unable to process the payment from the registered card on the due date. |
| 24 | What happens after a failed EMI payment attempt? | The system automatically retries the payment up to 4 times over 10 days. If all 4 attempts fail, the instalment arrangement is cancelled and the school will contact the parent directly to arrange an alternative payment method. |
| 25 | How can a user add multiple items to their payment basket? | Navigate to the Shop section from the Home Menu (not from the Attention Dashboard) and add each item to the cart individually. Once all items are added, select the cart icon (top right) and proceed to checkout to pay for all items in one transaction. (Note: items accessed via the Attention Dashboard can only be checked out one at a time.) |
| 26 | Why might a user not see items in the shop section? | Three possible reasons: (1) The payment item has not been assigned to the student by the school; (2) The final payment date for the item has already passed; (3) The item was limited to a set number of students and the maximum has already been reached. |
| 27 | How can a user remove unwanted items from their shop list? | Use the "Opt Out" feature in the Shop. This is available for optional items such as evening study, voluntary donations, and optional trips. Opting out removes the item from the Shop and records the parent as having opted out. |
|  | 🔹 Easypayments |  |
| 28 | Who should a user contact for issues related to Easypayments? | Users should contact Easypayments directly. Unique Schools does not manage or operate the Easypayments platform and is unable to assist with issues related to it. |
| 29 | Does Unique Schools have access to Easypayments systems? | No. Unique Schools does not have access to Easypayments systems or data. Any Easypayments-related queries must be directed to Easypayments support. |
|  | 🔹 App Profile Issues |  |
| 30 | What should a user do if a name is incorrectly spelled in the app profile? | Contact the school directly to have the name corrected in the school's administration system. Since Unique Schools cannot modify school records, the school must update the name, after which it will be reflected in the App. |
| 31 | What should a user do if they cannot see all their children in the app? | Contact the school to confirm that all children are correctly linked to the parent/guardian's account in the school's administration system. Alternatively, contact support@uniqueschools.ie who will liaise with the school to resolve the issue. |
